# Repositorio de ejercicios prácticos del ciclo de DAW para MPO

Este repositorio contiene una colección de ejercicios prácticos realizados durante el ciclo de Desarrollo de Aplicaciones Web (DAW) para el módulo de profesional optativo (MPO) que se usa para la ampliación en programación en Python.

Cada sesión tiene un directorio propio con todos los ejercicios ese dia.

## Descarga y uso

Puedes clonar este repositorio usando Git o descargarlo como un archivo ZIP. Para clonar el repositorio, usa el siguiente comando en tu terminal:

```bash
git clone https://github.com/jordicido/DAWMPOFeb26.git
```

Una vez clonado o descargado, puedes navegar por los directorios para encontrar los ejercicios correspondientes a cada sesión. Cada ejercicio está acompañado de su solución para que puedas comparar tu trabajo con la solución propuesta.

Para actualizar el repositorio con los últimos ejercicios, puedes usar el siguiente comando dentro del directorio del repositorio:

```bash
git pull origin main
```
